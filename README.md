# Task 6 - Password Strength Analysis

##  Introduction

This project is part of my Cyber Security Internship. In this task, I created different types of passwords and tested their strength using an online password checker tool.

## Objective

To learn how to create strong passwords and evaluate their strength for better security.

##  Tools Used

* PasswordMeter (Online Password Strength Checker)

##  Work Done

* Created multiple passwords (weak to strong)
* Tested each password using an online tool
* Recorded strength, score, and feedback
* Analyzed how password complexity affects security

##  Results

* Simple passwords are very weak and easy to guess
* Adding uppercase, lowercase, numbers, and symbols increases strength
* Longer passwords are more secure

## Common Attacks

* **Brute Force Attack**: Tries all possible combinations
* **Dictionary Attack**: Uses common words to guess passwords

##  Conclusion

From this task, I learned that strong passwords are essential for security. Using a mix of characters and longer length makes passwords harder to crack.
